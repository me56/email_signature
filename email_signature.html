美因基因个性邮箱签名

复制这些HTML代码

以腾讯网页版的企业邮箱为例,设置个性签名

登录邮箱 --> 点击设置 --> 常规 --> 个性签名 --> 添加个性签名 --> 给自己的签名命名 --> HTML --> 复制第3步的代码并粘贴 --> 保存

⑤给自己的签名命名 --> HTML --> 复制第3步的代码并粘贴 --> 保存

⑥写信时，选择自己设置的个性签名

以windows用户foxmail客户端为例,设置个性签名:

找到fox客户端的设置 --> 写邮件 --> 高级 --> 新建或删除(原来的) --> 点击右上角红框 --> 删除里面自带 样式 --> 把刚才复制的代码 粘贴过来 --> 确定 --> 勾选启用签名 -->确定.
